# TierForge

Crie e compartilhe **tier lists** com amigos — drag & drop, raridades de jogos, upload de imagens, banco de dados Supabase.

## ✨ Features
- 🏆 Hub público com todas as tier lists
- 🎮 Templates de raridades (Fortnite, MMORPGs) ou Tier Clássico
- 🔮 Itens quadrados ou esfera
- 👁 Nomes dos itens (do filename, editável)
- ↔️ Labels redimensionáveis
- 🔒 Proteção por senha (só edição)
- 💾 Banco de dados Supabase + Storage para imagens
